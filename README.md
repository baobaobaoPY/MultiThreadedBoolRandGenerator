## 编译器差异 Linux
**`g++`**  
**`-O3`的优化下，V1版本在我的设备中实现了160亿次布尔值生成仅需5s😎！**  
**`-O3`的优化下，V2版本在我的设备中实现了160亿次布尔值生成仅需3s🤯！**  
---
**`clang++`**  
**`-O3`的优化下，V1版本在我的设备中生成160亿次布尔值居然花费44s🤔？**  
**`-O3`的优化下，V2版本在我的设备中实现了160亿次布尔值生成仅需1s🤯！？**
